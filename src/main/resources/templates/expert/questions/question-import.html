Supported formats: CSV, Excel (.xlsx, .xls)

File Format Guide

CSV/Excel file should have the following columns:

  • questionText - The question text (required)
  • questionType - MULTIPLE_CHOICE, TRUE_FALSE, SHORT_ANSWER, ESSAY
  • points - Points for this question (default: 1)
  • options - For multiple choice: comma-separated options (e.g., "Option A,Option B,Option C")
  • correctAnswer - The correct answer or option index
  • explanation - Explanation for the answer (optional)
